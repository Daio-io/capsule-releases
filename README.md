# Capsule CLI Releases

Terminal podcast client for [Capsule](https://capsulefm.io).

## Install

```bash
brew tap Daio-io/capsule-releases
brew install capsule
```

## Update

```bash
brew update && brew upgrade capsule
```

## Requirements

- macOS (Intel or Apple Silicon)
- **ffmpeg** (optional, required for radio playback): `brew install ffmpeg`
